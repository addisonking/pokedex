# Pokedex Tracker

Local-first app to track regional & national Pok√©dex progress for gens 1‚Äì5.
In-app encounter panel with per-version locations + evolution hints; Serebii
is a fallback link inside that panel. Runs as a web app or a Tauri desktop
app; all progress lives in localStorage on the device.

## Commands

- `bun install` ‚Äî install deps
- `bun run dev` ‚Äî start dev server
- `bun run build` ‚Äî typecheck + production build (outputs to `dist/`)
- `bun run typecheck` ‚Äî `tsc --noEmit`
- `bun run lint` ‚Äî `biome check src`
- `bun run format` ‚Äî `biome format --write src`
- `bun run tauri dev` ‚Äî run desktop app (starts Vite dev server automatically)
- `bun run tauri build` ‚Äî build + bundle desktop app (requires Rust)
- `bun run generate` ‚Äî regenerate `src/data/*.json` from PokeAPI: 649 pokemon + 8 regional dexes + 649 encounter calls + 649 species + ~328 evolution chains (~1700 calls total)
- `bun run sprites` ‚Äî download sprites 1‚Äì649 into `public/sprites/` from PokeAPI sprites repo (idempotent)

## Stack

Vite + React 18 + TypeScript + Tailwind v4 + Biome. React Router (HashRouter, so
static hosts work without SPA fallback config). shadcn/ui components. Tauri v2
(`src-tauri/`) wraps the same build for desktop.

## Tauri

`src-tauri/tauri.conf.json` ‚Äî dev uses Vite at `localhost:5173`, release bundles
`dist/`. `tauri-plugin-opener` is registered; `src/lib/external-links.ts`
intercepts `target="_blank"` clicks inside the Tauri webview and opens them in
the system browser (webviews ignore `_blank` otherwise). No-op in a regular
browser.

## State

All progress in `localStorage` under key `pokedex`: `{ playthroughs, progress }`.
- `playthroughs` ‚Äî one per save file (id, gameId, version, name, mode, createdAt, setupDone)
- `progress` ‚Äî per playthrough id, sparse map of pokemonId ‚Üí `{ s: 1|2, t?: timestamp }`
`src/lib/storage.tsx` ‚Äî ProgressProvider context, validates on load, persists on
every change. `useProgress()` hook. Settings menu has JSON export/import for
backup and a full reset.

## Save importing (gen 1-5)

`src/lib/save-parser/` ‚Äî parses .sav files to extract Pok√©dex seen/owned bitfields.
`importSave(buf: ArrayBuffer)` ‚Üí `{ gameId, version, name, seen[], caught[] } | null`.
Auto-detects gen via file size + checksum/signature validation:
- Gen 1 (32KB): 8-bit inverted sum checksum, no crypto
- Gen 2 (32KB): u16 checksum, two slots, no crypto
- Gen 3 (128KB): plaintext 14-section layout, signature 0x08012025, section 0 = dex
- Gen 4 (512KB): plaintext blocks, magic 0x20060623, CRC16-CCITT, Zukan4 structure
- Gen 5 (512KB): plaintext blocks, CRC16-CCITT footer validation, Zukan5 structure
Offsets from PKHeX source (kwsch/PKHeX). Gen 4-5 distinguished by footer magic vs CRC.
Dex bitfields: LSB-first, bit N = national ID N+1. Home page "Import .sav" button
calls importSave ‚Üí addPlaythrough ‚Üí bulkSet(seen,1) ‚Üí bulkSet(caught,2) ‚Üí
completeSetup (imported marks are baseline, no `t` timestamp).

## Data

`src/data/pokemon.json` ‚Äî national 1‚Äì649: `{ id, name, types, gen }`.
`src/data/games.json` ‚Äî 8 games, each with `serebiiSuffix` + `dex: [{ r, n }]`
(`r` = regional number, `n` = national id). Generated by `scripts/generate-data.ts`.
`src/data/encounters/{version}.json` ‚Äî 20 files (red ‚Ä¶ white-2), lazy-loaded by
`src/lib/encounters.ts` via `import.meta.glob`. Per-version map of national id ‚Üí
`[{ loc, method, chance, min, max, cond? }]`. `loc` prettified from location_area
slug (region prefix stripped, `-area` stripped, title-cased). Aggregate per
`(loc, method, sortedConds)`: sum chance (cap 100), merge level range. ~2.4 MB
total, code-split by Vite so base bundle unaffected.
`src/data/evolutions.json` ‚Äî small (~20 KB) statically-imported file, keyed by
evolved id: `{ from, trigger, level?, item?, note? }`. `note` is a compact human
string for happiness/time/held-item/known-move/beauty/gender/location conditions.
Types in `src/types.ts`: `EncounterRow`, `VersionEncounters`, `Evolution`.
`src/lib/encounters.ts` ‚Äî `loadEncounters(version)` (promise-cached, missing file
‚Üí `{}`), `METHOD_LABELS`/`CONDITION_LABELS` slug‚Üílabel dicts with title-case
fallback, `evolutionHint(id)` ‚Üí "Evolve X ‚Äî Lv 16" / "‚Ä¶ ‚Äî Trade" / "‚Ä¶ ‚Äî Fire Stone".
Serebii location URL (fallback in dialog): `serebii.net/pokedex-{suffix}/location/{NNN}.shtml#{anchor}`
‚Äî deep-links to the in-depth location page for the playthrough's version.
Version‚Üíanchor map in `src/lib/serebii.ts` (e.g. gold‚Üíg, ruby‚Üíruby, heartgold‚Üíhg).
Gen 1 suffix is empty. `src/components/EncounterDialog.tsx` renders the in-app panel;
footer Serebii ‚Üó uses `target="_blank"` (Tauri routes to system browser via
`src/lib/external-links.ts`).
`public/sprites/{id}.png` ‚Äî pixel sprites for 1‚Äì649, vendored locally (no runtime dep on
GitHub). Downloaded by `scripts/download-sprites.ts`. `PokemonCard` serves them at
`/sprites/{id}.png`.
